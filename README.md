## Install

#### Install all faker packages.

npm
```
npm install @fakerjs/faker --save-dev
```

yarn
```
yarn add @fakerjs/faker -D
```

## Usage

```js
import faker from '@fakerjs/faker';

faker().animal();
//=> Snow Leopard

faker().gender();
//=> Female

faker().profession();
//=> Software Engineer

// etc...
```
